# 画面遷移図

[← 要件定義書に戻る](../requirements.md)

---

## 1. 全体の画面遷移

```mermaid
flowchart LR
    S01["S-01\nログイン画面"]
    S02["S-02\nユーザー登録画面"]
    S03["S-03\nタイムライン画面"]
    S04["S-04\n投稿詳細・コメント画面"]
    S05["S-05\nプロフィール画面"]
    S06["S-06\nユーザー検索画面"]

    Start([未ログイン]) --> S01
    Start --> S02
    S01 -- ログイン成功 --> S03
    S02 -- 登録完了 --> S03
    S01 -- 「登録はこちら」リンク --> S02
    S02 -- 「ログインはこちら」リンク --> S01

    S03 -- 投稿カードをクリック --> S04
    S03 -- ユーザー名をクリック --> S05
    S03 -- ナビ「検索」 --> S06
    S03 -- ナビ「プロフィール」 --> S05

    S04 -- 投稿者名をクリック --> S05
    S04 -- 「戻る」ボタン --> S03

    S05 -- 「戻る」ボタン --> S03
    S05 -- 投稿カードをクリック --> S04

    S06 -- ユーザー名をクリック --> S05
    S06 -- 「戻る」ボタン --> S03
```

---

## 2. 認証フロー

```mermaid
flowchart TD
    A([リクエスト]) --> B{ログイン済み?}
    B -- はい --> C[対象画面を表示]
    B -- いいえ --> D[S-01 ログイン画面へリダイレクト]
    D --> E[ログイン成功]
    E --> C
```

---

## 3. 画面一覧

| 画面 ID | 画面名 | 認証必要 | URL | 主な遷移元 |
| --- | --- | --- | --- | --- |
| S-01 | ログイン画面 | 不要 | `/login` | 未ログイン時・ログアウト後 |
| S-02 | ユーザー登録画面 | 不要 | `/register` | S-01 のリンク |
| S-03 | タイムライン画面 | 必要 | `/` | ログイン後・ナビ |
| S-04 | 投稿詳細・コメント画面 | 必要 | `/posts/:id` | S-03 / S-05 の投稿カード |
| S-05 | プロフィール画面 | 必要 | `/users/:id` | ユーザー名クリック・ナビ |
| S-06 | ユーザー検索画面 | 必要 | `/search` | ナビの検索アイコン |
