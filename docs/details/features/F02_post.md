# F-03 投稿（作成・編集・削除）

[← 要件定義書に戻る](../../requirements.md)

---

## 1. 概要

ログインユーザーがテキスト（280文字以内）と任意の画像を投稿できる。
投稿の編集・削除は投稿者本人のみ可能。画像は AWS S3 に保存する。

---

## 2. 対象画面

| 画面 ID | 画面名 |
| --- | --- |
| S-03 | タイムライン画面（投稿フォーム） |
| S-04 | 投稿詳細・コメント画面（編集・削除） |

---

## 3. 業務フロー

```mermaid
flowchart TD
    A([投稿操作開始]) --> B{操作種別}
    B -- 作成 --> C[テキスト入力\n最大280文字]
    C --> D{画像添付?}
    D -- あり --> E[画像選択 → S3 アップロード]
    D -- なし --> F[「投稿」ボタン押下]
    E --> F
    F --> G{入力チェック}
    G -- NG --> H[エラーメッセージ]
    H --> C
    G -- OK --> I[posts テーブルに保存]
    I --> J[タイムラインに反映]

    B -- 編集 --> K[本人確認]
    K --> L[フォームにテキスト・画像を表示]
    L --> M[修正して「保存」ボタン押下]
    M --> N{入力チェック}
    N -- NG --> H
    N -- OK --> O[posts テーブルを更新\nupdated_at を更新]
    O --> J

    B -- 削除 --> P[本人確認]
    P --> Q[確認ダイアログ表示]
    Q --> R[「削除」ボタン押下]
    R --> S[posts テーブルから削除\nS3 画像も削除]
    S --> J
```

## 5. IPO

### 投稿作成

| 項目 | 内容 |
| --- | --- |
| 入力 | テキスト（必須）・画像ファイル（任意） |
| 処理 | 入力チェック → 画像があれば S3 アップロード → posts テーブルに保存 |
| 出力 | 作成した投稿オブジェクト / エラーメッセージ |

### 投稿編集

| 項目 | 内容 |
| --- | --- |
| 入力 | テキスト（必須）・画像ファイル（任意）・投稿 ID |
| 処理 | 本人確認 → 入力チェック → 画像差し替えがあれば旧 S3 オブジェクト削除 → posts テーブル更新 |
| 出力 | 更新した投稿オブジェクト / エラーメッセージ |

### 投稿削除

| 項目 | 内容 |
| --- | --- |
| 入力 | 投稿 ID |
| 処理 | 本人確認 → S3 画像オブジェクト削除 → posts テーブルから削除 |
| 出力 | 204 No Content / エラーメッセージ |

---

## 6. 入力チェック仕様

| 項目 | 必須 | 形式・制約 | エラーメッセージ |
| --- | --- | --- | --- |
| テキスト | ○ | 1〜280文字 | 「投稿テキストは1〜280文字で入力してください」 |
| 画像 | — | JPEG / PNG / GIF。5MB 以内 | 「画像は JPEG・PNG・GIF 形式、5MB 以内でアップロードしてください」 |

---

## 7. エラーメッセージ

| コード | メッセージ | 発生条件 | 重要度 |
| --- | --- | --- | --- |
| E-010 | 投稿テキストは1〜280文字で入力してください | テキストが空または281文字以上 | E |
| E-011 | 画像は JPEG・PNG・GIF 形式、5MB 以内でアップロードしてください | 不正な画像形式またはサイズ超過 | E |
| E-012 | この投稿を操作する権限がありません | 他ユーザーの投稿を編集・削除しようとした | E |
| E-013 | 投稿が見つかりません | 存在しない投稿 ID を指定 | E |

## 8. API エンドポイント

| メソッド | パス | 説明 |
| --- | --- | --- |
| POST | `/api/posts` | 投稿作成（multipart/form-data） |
| GET | `/api/posts/{id}` | 投稿詳細取得 |
| PUT | `/api/posts/{id}` | 投稿編集（本人のみ） |
| DELETE | `/api/posts/{id}` | 投稿削除（本人のみ） |
